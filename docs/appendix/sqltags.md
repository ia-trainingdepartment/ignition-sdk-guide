## Tag Structure
SQLTags are fundamentally defined by a set of properties, contained in the TagProp enum. The values of this enum are defined below:

| Property   	| Type  	| Default Value  	| Notes  	|
|---	|---	|---	|---	|
| Value  	| Object  	|   	|   	|
| Name  	| String  	|   	|   	|
| Quality  	| DataQuality  	| DataQuality.Unknown  	|   	|
| Enabled  	| Boolean  	| true  	|   	|
| AccessRights  	| AccessRightsType  	| AccessRightsType.Read_Write  	| If custom, the tag's permission model will be consulted for the allowed rights  	|
| OPCServer  	| String  	|   	| Only used when the tag's type is OPC  	|
